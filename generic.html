AstroKit Learning Modules

Jump into curated lessons designed for Filipino classrooms, AV clubs, and self-learners. Each module includes explanations, visuals, activities, and worksheets.

01 // Introduction to Radio Astronomy
Learn how radio telescopes work, why we listen to the sky, and how electromagnetic waves reveal hidden cosmic structures.
02 // The Hydrogen Line (1420 MHz)
Explore the famous 21-cm line, how it maps the Milky Way, and why your own antenna can detect it.
03 // Building a Radio Telescope
Step-by-step guide to your circular patch feed, LNA, RTL-SDR setup, calibration, and safety.
04 // Data Collection & Analysis
Learn how to capture signals, read graphs, filter noise, and analyze the hydrogen line with confidence.
05 // PH Classroom Astronomy
Activities, worksheets, and local-context lessons connecting space science with the Philippines.
